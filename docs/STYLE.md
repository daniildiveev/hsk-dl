# Код-стайл (кратко)

- Модули: `Core.*` — низкоуровневые; без побочных эффектов где возможно.
- Экспортируй минимально необходимое API.
- Имена: функции — `camelCase`, типы/конструкторы — `CamelCase`.
- Ошибки: только человекочитаемые сообщения, без "pattern match failure".
- Вьюхи не аллоцируют; материализация — через `contiguous`.
- Тесты: property-тесты там, где можно (broadcast/reshape), unit — для ошибок.
